- 업비트, 코인원, 코빗은 기본 스크립트 추가 완료
- 업비트/코인원/코빗은 잔고조회와 Kakao 전송 경로 확인 완료
- 새 거래소 또는 계정이 추가되면 이 문서를 같은 표 형식으로 갱신
