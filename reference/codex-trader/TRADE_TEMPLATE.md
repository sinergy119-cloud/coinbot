# 코인 거래 템플릿

아래 템플릿은 사용자가 `코인 거래 템플릿 보여줘` 또는 `템플릿`을 요청할 때 기본으로 회신하는 내용입니다.

```text
[코인 거래 프롬프트 (이벤트 실적용)]

-----------------------------------------------------------
- 정보 소스 : 코인 API 리스트.txt 파일
1. [★중요] 거래 요청 내용
    1) 거래소 : 고팍스
    2) 이름 : 전체
    3) 코인명 : DEGEN UP 
    4) 매수 : 시장가 51,000원 
    5) 매도 : 시장가 전량 매도
    6) 추가 반복 횟수 : 없음 
2. 거래 요청의 리스트를 보여준다. (거래소, 이름, 주문 내용, 잔고(KRW), 가능/불가능)
3. 진행 여부를 물어본다.
4. 최종 결과 보여준다. (거래소, 이름, 코인명, 거래요청 내용, 결과: 성공/실패, 체결금액)
5. 잔고를 보여준다. (거래소, 이름, 자산 현황)
-----------------------------------------------------------
```

운영 규칙:
- 템플릿 변경 요청이 있으면 이 파일 내용을 기준으로 업데이트합니다.
- 이후 `코인 거래 템플릿 보여줘`, `거래 템플릿`, `템플릿` 요청에는 이 파일의 최신 내용을 코드블록으로 보여줍니다.
