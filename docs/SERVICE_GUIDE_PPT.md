## [슬라이드 2] 코인 에어드랍 이벤트란?

**핵심 메시지:**
거래소가 특정 기간 동안 거래에 참여한 사용자에게
코인 상금을 N빵으로 나눠주는 이벤트

**참여 방법 — "사팔" 거래:**

| 거래 조건 | 실제 거래 방법 | 거래소 예시 |
|-----------|----------------|-------------|
| 일일 1만원 이상 | 5,100원 매수 → 5,100원 매도 | 코빗 CROSS |
| 일일 10만원 이상 | 51,000원 매수 → 51,000원 매도 | 고팍스 UP |

※ 소액 매수 후 즉시 매도 = 조건 충족. 매일 반복 필요.

## [슬라이드 3] MyCoinBot이란?

**핵심 메시지:**
이 반복 거래를 매일 자동으로 대신 해주는 서비스

**흐름도:**
① 계정 등록 → ② 스케줄 설정 → ③ 자동 사팔 → ✓ 이벤트 참여 완료

**지원 거래소 (5대):**
빗썸 / 업비트 / 코인원 / 코빗 / 고팍스

---

## [슬라이드 4] 서비스 특징

**가족·지인 계정 대행**
본인 외 여러 계정을 한 번에 운영 가능
가족계정 번거로움 해소

**스케줄 자동화**
원하는 시간에 자동 실행
매일 수동으로 할 필요 없음

**시장가 즉시 체결**
매수·매도 모두 시장가로 즉시 처리
CYCLE(사팔) / BUY(매수) / SELL(매도) 선택

**보안 설계**
API Key는 AES-256 암호화 저장
입출금 권한 없는 키만 사용

---

## [슬라이드 5] 시작하는 방법

**서비스 이용 전 준비사항 3가지:**

① 거래소 계정 개설
   빗썸 / 업비트 / 코인원 / 코빗 / 고팍스 중 선택하여 가입

② API Key 발급
   거래소 마이페이지 → API 발급
   반드시 "입출금 권한 제외" 설정 (보안)

③ 거래 자금 입금
   거래 금액 + 수수료 여유분 입금
   최소 거래 금액: 5,100원

신규 가입 이벤트
→ 무료 체험 혜택 제공

## [슬라이드 7] 유의사항

- 코인원 API 제한: API 거래량에 제한이 있어 대량 반복 거래 불가
- 이벤트 신청 직접 필요: 거래소 이벤트 신청은 사용자가 직접 해야 함
- 시장가 거래 손실: 매수/매도 시 스프레드로 인한 소액 손실 가능
- API 보안 필수: 입출금 권한 없는 API Key만 사용 권장
