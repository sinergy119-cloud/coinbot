# MyCoinBot 서비스 소개

> 코인 에어드랍 이벤트 자동 참여 서비스

---

## 1. 코인 에어드랍 이벤트란?

거래소가 특정 기간 동안 거래에 참여한 사용자에게 
**코인 상금을 N빵으로 나눠주는 이벤트**입니다.

### 참여 방법 (사팔 거래)

| 거래 조건 | 거래 방법 |
|-----------|-----------|
| 일일 **1만원 이상** 거래 | 5,100원 매수 → 5,100원 매도 (1회 사팔) |
| 일일 **10만원 이상** 거래 | 51,000원 매수 → 51,000원 매도 (1회 사팔) |

> 소액으로 매수 후 즉시 매도하면 조건 충족

### 코인 에어드랍 이벤트
- 빗썸, 업비트, 코인원, 코빗, 고팍스에서 다양한 이벤트 운영 중

---

## 2. MyCoinBot이란?

여러 계정의 반복 거래를 **스케줄 한 번으로 자동 실행하는 서비스**입니다.

```
[사용자] 거래소 계정 + API Key 등록
      ↓
[MyCoinBot] 매일 정해진 시간에 자동 사팔 실행
      ↓
[결과] 에어드랍 이벤트 자동 참여 완료
```

### 지원 거래소 (5대)
| 거래소 | 비고 |
|--------|------|
| 빗썸 | ✅ |
| 업비트 | ✅ |
| 코인원 | ✅ |
| 코빗 | ✅ |
| 고팍스 | ✅ |

---

## 3. 서비스 특징

### 개인 대행 가능
- 본인 계정 외 **가족·지인 계정도 함께 운영** 가능
- 여러 계정을 한 번에 자동 처리

### 자동화 거래
- 시장가 자동 매수·매도 (CYCLE: 사고팔기 / BUY: 매수만 / SELL: 매도만)
- 스케줄 설정으로 원하는 시간에 자동 실행

### 관리자 위임
- 거래 실행이 어려운 경우 **관리자에게 운영을 맡길 수 있음**
- 위임 ON/OFF는 언제든 본인이 직접 설정
- 위임 중에도 본인 계정으로 직접 거래 가능

---

## 4. 시작 전 준비사항

서비스 이용을 위해 아래 3가지가 필요합니다.

| 순서 | 항목 | 설명 |
|------|------|------|
| ① | **거래소 가입** | 빗썸 / 업비트 / 코인원 / 코빗 / 고팍스 |
| ② | **API Key 발급** | 각 거래소 마이페이지에서 발급 |
| ③ | **거래 자금 입금** | 거래 금액 + 수수료 여유분 |

> ⚠ API Key는 PC에서만 발급 가능 (모바일 불가)

### 자금 예시

| 거래 실적 목표 | 입금 금액 | 사팔 금액 |
|-------------|----------|----------|
| 1만원 이상 | **6,000원** 정도 | 5,100원 사팔 |
| 10만원 이상 | **60,000원** 정도 | 51,000원 사팔 |

---


## 5. 유의사항

| 항목 | 내용 |
|------|------|
| 이벤트 신청 | 이벤트 신청이 필요한 경우 사용자가 직접 해야 함 |
| 시장가 거래 | 매수/매도 시 시세에 따라 소액 손실 가능 |
| API 미허용 | 거래소에서 API 거래를 허용하지 않는 코인은 직접 거래 필요 |
