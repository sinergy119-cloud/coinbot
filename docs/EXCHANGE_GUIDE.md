### API Key 발급

- **메뉴:** PC 로그인 → 우측 상단 사람 아이콘 → 설정 → API 관리
- **발급 절차:**

```
① API 관리 페이지에서 "API 활성 항목 선택" 체크
② 휴대폰 SMS 인증
③ 보안비밀번호 입력
④ [API KEY 생성] 클릭
⑤ Connect Key(Access Key)와 Secret Key 복사/저장
   → ⚠ Secret Key는 이후 재확인 불가
⑥ [활성화] 클릭 → 이메일 승인 링크 클릭
```

- **권한 설정:**

| 권한 | 설정 |
|------|------|
| 회원지갑 정보 | ✅ ON |
| 주문내역 | ✅ ON |
| 회원거래내역 | ✅ ON |
| 매수/매도 주문 | ✅ ON |
| 출금 관련 | ❌ 반드시 OFF |

> ⚠ IP 허용 목록에 **43.203.100.239** 등록 필수

### API Key 발급

- **메뉴:** PC 로그인 → 마이페이지 → Open API 관리
- **발급 절차:**

```
① Open API 관리 페이지 접속
② 권한 선택 (아래 참고)
③ 개인정보 수집 동의
④ 2채널 인증(2FA) 본인 확인
⑤ Access Key와 Secret Key 복사/저장
   → ⚠ Secret Key는 최초 발급 화면에서만 확인 가능
```

- **권한 설정:**

| 권한 | 설정 |
|------|------|
| 자산 조회 | ✅ ON |
| 주문 | ✅ ON |
| 출금 | ❌ 반드시 OFF |

> ⚠ IP 주소 등록란에 **43.203.100.239** 입력 (최대 10개)
