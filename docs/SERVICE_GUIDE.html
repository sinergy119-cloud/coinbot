STEP 1

💰 코인 에어드랍 이벤트란?

거래소가 특정 기간 동안 거래에 참여한 사용자에게
코인 상금을 N빵으로 나눠주는 이벤트입니다.

참여 방법 — "사팔" 거래

거래 조건실제 거래 방법거래소 예시
일일 1만원 이상 5,100원 매수 → 5,100원 매도 코빗 CROSS
일일 10만원 이상 51,000원 매수 → 51,000원 매도 고팍스 UP

※ 소액 매수 후 즉시 매도 = 조건 충족. 매일 반복 필요.

⚡ 서비스 특징

👨‍👩‍👧‍👦

가족·지인 계정 대행

본인 외 여러 계정을 한 번에 운영 가능. 가족계정 번거로움 해소.

스케줄 자동화

원하는 시간에 자동 실행. 매일 수동으로 할 필요 없음.

⚡

시장가 즉시 체결

매수·매도 모두 시장가로 즉시 처리. CYCLE / BUY / SELL 선택.

🔒

보안 설계

API Key는 AES-256 암호화 저장. 입출금 권한 없는 키만 사용.

⚠️ 한계점 및 유의사항

  • 코인원 API 제한 — API 거래량에 제한이 있어 대량 반복 거래 불가
  • 이벤트 신청 직접 필요 — 거래소 이벤트 신청은 사용자가 직접 해야 함 (자동화 불가)
  • 시장가 거래 손실 — 매수/매도 시 스프레드로 인한 소액 손실 발생 가능
